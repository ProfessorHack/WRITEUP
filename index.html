Artifact (BlackHat Qual 2024)

Opening the hive with registry explorer:

We need to find an executable file that is some type of malware. By opening the following path, we can view the running time and name of the file:

HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache\AppCompatCache

(This technique is derived from the TryHackMe Windows Forensics 1 Room.)

At this location, two executable files (files.exe) are shown. One of them is:

DeadPotato.Net4.exe

The Flag:

BHFlagY{DeadPotato-NET4.exe_09/08/2024_22:42:13}
